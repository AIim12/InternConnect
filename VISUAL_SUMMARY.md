## рҹҺҜ User Workflows

### Student Workflow:
```
1. Land on Home Page
   в”ңв”Җ See Login Form on Right
   в””в”Җ Sign In

2. Go to Find Jobs
   в”ңв”Җ Adjust Filters (price, keywords, skills)
   в”ңв”Җ See matching jobs with match %
   в””в”Җ Click Apply

3. My Applications
   в”ңв”Җ See status of each application
   в”ңв”Җ If offer received, see message
   в””в”Җ Accept or decline

4. My Profile
   в”ңв”Җ Upload photo
   в”ңв”Җ Add Python + React (predefined)
   в”ңв”Җ Add "Leadership" (custom)
   в””в”Җ Save

5. After 10 min inactive
   в”ңв”Җ Automatic logout
   в”ңв”Җ Alert shown
   в””в”Җ Back to home page
```

### Employer Workflow:
```
1. Land on Home Page
   в”ңв”Җ See Login Form
   в”ңв”Җ Select "Employer" role
   в””в”Җ Sign Up

2. Go to Company Profile
   в”ңв”Җ Enter "TechCorp"
   в”ңв”Җ Enter company bio
   в””в”Җ Save

3. Post Internship
   в”ңв”Җ Title: "Backend Dev"
   в”ңв”Җ Description: "Build APIs"
   в”ңв”Җ Skills: Python, SQL
   в”ңв”Җ Rate: $20/hr
   в”ңв”Җ Hours: 40/week
   в”ңв”Җ Remote: Yes
   в””в”Җ Publish

4. View Applications
   в”ңв”Җ See candidate list
   в”ңв”Җ Review skills
   в”ңв”Җ Click "Accept"
   в”ңв”Җ Type offer message
   в”ңв”Җ Send

5. Application sent to student
```

## рҹ”Ң API Endpoints Overview

```
USER MANAGEMENT:
  POST   /auth/register
  POST   /auth/login
  GET    /auth/me

STUDENT:
  PUT    /auth/profile                    Get/Update profile
  GET    /auth/profile
  POST   /auth/internships/{id}/apply     Apply to job
  GET    /auth/applications/me             View my applications

EMPLOYER (NEW):
  PUT    /auth/employer/profile           Manage company
  GET    /auth/employer/profile
  POST   /auth/internships                Create job
  PATCH  /auth/internships/{id}           Update job (NEW)
  DELETE /auth/internships/{id}           Delete job (NEW)
  GET    /auth/internships/my             View my jobs
  GET    /auth/internships/{id}/applicants View applicants

JOBS:
  GET    /auth/internships                Browse all (with filters)

APPLICATIONS:
  GET    /auth/internships/{id}/applicants
  PATCH  /auth/internships/{id}/applicants/{email}  Update status
```
